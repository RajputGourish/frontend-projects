# 🔗 BitLinks – URL Shortener

BitLinks is a **URL shortening web application** that allows users to convert long URLs into short, shareable links.

The project is built using **Next.js and MongoDB**, demonstrating how modern full stack applications can generate, store, and redirect shortened URLs efficiently.

---

## 🚀 Features

- Generate short links from long URLs
- Store links in MongoDB database
- Redirect short URLs to the original website
- Simple and responsive UI
- Built using modern **Next.js full stack architecture**

---

## 🛠 Tech Stack

- **Next.js**
- **React**
- **Tailwind CSS**
- **MongoDB**
- **Node.js (Next.js API Routes)**
- **JavaScript (ES6+)**

---

## ⚙️ How It Works

1. User enters a **long URL** in the input field.
2. The application generates a **unique short code**.
3. The long URL and short code are stored in **MongoDB**.
4. When someone opens the short link, the app **redirects them to the original URL**.
