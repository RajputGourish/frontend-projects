# 🐦 X (Twitter) UI Clone — Tailwind CSS

A UI clone of the X (formerly Twitter) interface built using **HTML and Tailwind CSS**.

This project focuses on recreating a modern social media layout while practicing utility-first styling and responsive design using Tailwind CSS.

---

## 🚀 Features

- Sidebar navigation layout
- Feed section UI
- Profile & trending panels
- Responsive layout structure
- Modern social media interface design

---

## 🛠 Tech Stack

- HTML5
- Tailwind CSS

---

## 🎯 Learning Objective

Through this project, I practiced:

- Utility-first CSS workflow
- Responsive layouts with Tailwind
- Real-world UI recreation
- Faster styling using Tailwind classes
- Layout structuring for large interfaces

---

## ⚠️ Note

This project is **UI only** and does not include backend functionality or authentication features.

---

## ▶ How to Run

Open `index.html` in your browser  
or run using Live Server.
