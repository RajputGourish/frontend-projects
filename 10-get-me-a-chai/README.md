# ☕ Get Me A Chai

A **Patreon-style creator support platform** where users can support creators by sending small payments — like buying them a chai ☕.

This project is inspired by platforms such as Patreon and Buy Me A Coffee, allowing fans to financially support creators.

---

## 🚀 Features

- Creator profile page
- Support creators with payments
- Razorpay payment gateway integration
- Clean and responsive UI
- Real-time payment handling
- Modern Next.js application structure

---

## 🛠 Tech Stack

- **Next.js**
- **React**
- **Tailwind CSS**
- **Razorpay Payment Gateway**
- **JavaScript (ES6+)**
