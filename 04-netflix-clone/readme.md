# 🎬 Netflix India – Landing Page Clone

A responsive clone of the Netflix India landing page built using **HTML5 and CSS3**.

This project recreates the UI structure and design of Netflix’s official homepage for learning and practice purposes.


---

## 📌 Features

- 🔥 Hero section with background overlay
- 🎥 Video inside TV frame (like original Netflix)
- 📱 Responsive layout
- ❓ FAQ section UI
- 🎨 Netflix-style buttons and typography
- 🧾 Structured footer section

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Flexbox
- Positioning & Overlay Techniques
