# 🔐 Password Manager (MongoDB Version)

A full stack **Password Manager application** built with **React, Node.js, and MongoDB**.

This version extends the previous LocalStorage-based password manager by introducing a **database backend**, allowing passwords to be stored and retrieved from **MongoDB**.

---

## 🚀 Features

- Add new website credentials
- Save username and password securely
- View saved credentials
- Delete stored passwords
- Persistent storage using MongoDB
- Responsive UI using Tailwind CSS

---

## 🛠 Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

---

## 📚 What I Practiced in This Project

- Connecting frontend with backend API
- Performing CRUD operations
- Storing data in MongoDB
- Managing application state in React
- Handling asynchronous requests using fetch/axios

## ⚡ How to Run the Project

### 1️⃣ Install dependencies

Frontend

```
cd client
npm install
```

Backend

```
cd server
npm install
```

---

### 2️⃣ Start the backend

```
node server.js
```

---

### 3️⃣ Start the frontend

```
npm run dev
```
