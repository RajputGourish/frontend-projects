# ✅ To-Do List App — React + Tailwind CSS

A modern and interactive To-Do List application built using **React**, **Tailwind CSS**, and **LocalStorage**.

This project focuses on building a functional task management application using component-based architecture and persistent client-side storage.

---

## 🚀 Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✔ Mark tasks as completed
- ✏ Edit existing tasks
- 💾 Persistent data using LocalStorage
- ⚡ Fast and responsive UI
- 🎨 Styled using Tailwind CSS

---

## 🛠 Tech Stack

- React (Functional Components & Hooks)
- Tailwind CSS
- JavaScript (ES6+)
- LocalStorage API

---

## 📚 Concepts Practiced

- React functional components
- useState hook for state management
- useEffect for syncing LocalStorage
- Handling form inputs
- Conditional rendering
- Component re-rendering
- Utility-first styling with Tailwind CSS
