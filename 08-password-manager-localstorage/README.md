# 🔐 Password Manager App (LocalStorage Version)

A secure and responsive Password Manager built using **React and Tailwind CSS**.

This version stores user credentials in **LocalStorage**, allowing data persistence without a backend.

---

## 🚀 Features

- Add new website credentials
- Save website name, username/email, and password
- Show / hide password functionality
- Delete saved credentials
- Persistent data using LocalStorage
- Responsive and clean UI using Tailwind CSS

---

## 🛠 Tech Stack

- React.js (Functional Components)
- useState Hook
- LocalStorage API
- Tailwind CSS
- JavaScript (ES6+)

---

## 📂 Project Purpose

This project was built to:

- Practice React state management
- Work with browser LocalStorage
- Build a real-world utility application
- Improve UI design using Tailwind CSS
- Understand data persistence without backend

---

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/RajputGourish/frontend-projects.git
   ```

2. Navigate to the project folder:
   ```bash
   cd 08-password-manager-localstorage
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start development server:
   ```bash
   npm start
   ```

---

## 📌 Note

⚠️ This version uses LocalStorage for learning purposes.  
Passwords are stored in the browser and are not encrypted.  

A future version will include MongoDB backend integration for better security.
