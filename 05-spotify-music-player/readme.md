# 🎵 Spotify Music Player Clone

A functional Spotify-inspired music player built using **HTML, CSS, and JavaScript**.

This project recreates a modern music streaming interface with real audio playback functionality, dynamic playlists, album loading, and player controls.

---

## 🚀 Features

- 🎧 Play / Pause music
- ⏭ Next & Previous track navigation
- 📂 Dynamic album & song loading
- ⏱ Live song duration & progress bar
- 🔊 Volume control & mute option
- 📱 Sidebar navigation menu
- 🎵 Interactive music playlist

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
- Audio API

---

## 📚 Concepts Practiced

- DOM Manipulation
- Async JavaScript (`fetch`, `async/await`)
- Event Handling
- Audio Control using JavaScript
- Dynamic UI Rendering
- File & Folder Data Handling

---
